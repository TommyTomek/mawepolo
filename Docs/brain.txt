## 4. HARD RULES (enforce every session)
- **SSR:** Never use `window/document/localStorage/navigator` without `isPlatformBrowser()` guard
- **Components:** Standalone only · `OnPush` CD · no inline templates/styles · no business logic
- **i18n:** All text via translation keys (dot notation) · no hardcoded strings anywhere
- **Routing:** `provideRouter` in `app.config.ts` · `loadComponent` lazy loading · no language URL prefixes
- **TypeScript:** Never use `any` · prefer interfaces · `readonly` where appropriate
- **RxJS:** `switchMap` over nesting · `takeUntilDestroyed()` for cleanup · `async` pipe in templates
- **Forbidden:** `console.log` · magic strings · direct DOM access · deprecated Angular APIs · third-party UI libs (unless approved)
- **Allowed libs:** Angular built-ins, RxJS, Angular Universal, ngx-translate only

---

## 5. KEY FILE PATHS
| File | Purpose |
|---|---|
| `src/app/core/i18n/translation-loader.ts` | SSR-safe HTTP translation loader |
| `src/app/core/i18n/language.service.ts` | Language state, `setLanguage()`, localStorage (browser only) |
| `src/app/app.config.ts` | `provideRouter`, providers |
| `public/assets/i18n/{en,pl,it}.json` | Translation files (dot-notation keys) |

**Translation key patterns:**
- `regions.<id>.description`
- `cities.<id>.description`
- `home.title`, `navbar.login`, etc.

## 7. CONTENT RULES
- Region descriptions: 3–5 sentences, atmospheric, no clichés, no invented facts
- City descriptions: 3–6 sentences, present tense, narrative (not bullet facts)
- All images: `.webp` · hero images `1920×1080+` · city icons `512×512`
